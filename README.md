Allows for setting the media resolution parameter for the Gemini 3 series via a user friendly ui element in TypingMind, low through high and ultra (which is added for a given attachment per Gemini docs).

# GeminiVideoFPS.js

Allows for the setting of the video fps parameter via a ui element in TypingMind. See the Google Gemini 3 Docs for more information.  
Installation link:  
*https://cdn.jsdelivr.net/gh/IsaiahDePorto/TypingMindExtensions@refs/heads/main/GeminiVideoFPS.js*

# GeolocationSandboxUnlock.js

Allows for the usage of location based plugins in TypingMind which would otherwise give errors due to the iframe environment plugins are run in. Note: will still require location permissions for TypingMind in your browser and a separate location based plugin.  
Installation link:  
*https://cdn.jsdelivr.net/gh/IsaiahDePorto/TypingMindExtensions@refs/heads/main/GeolocationSandboxUnlock.js*
